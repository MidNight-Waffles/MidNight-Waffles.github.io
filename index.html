Welcome to Our App

This app is based on Mark Reiter, runs the Code Day Chicago event, and his awesome Waffle making skills. He makse his waffles beautifully and he makes them at Midnight.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Designing the Game

The app was created using Java and the characters, adn enemies, were created using Photo Shop. It took a lot of time to think of ideas needed to make it a quality game with a lot of items, power ups, and power downs. We had a team of 6 with each person doing a different part of the Project. Sometimes 2-3 people worked on one part, but than went back to doing their own thing to help the project, again.

The Power ups and Downs were a creative activity throughout the whole group. Each one took time to develope a special ability for each item that can help you and go against you.
